Welcome to Visual Analytics

This project aims at creating a tool to easily visually explore data stored into a NoSQL database through your web browser. Here's how to setup the server:

$ git pull va
$ npm install
$ node app

Database driven

The data gets pulled from the web and is stored in a MongoDB database. Next, the data is pulled from the database, passed through a crossfilter and visually laid out through the dc and d3 Javascript libraries. The slice and dice and interactivity of the charts is made possible by the crossfilter Javascript module.
